## [0.1.0] - 2025-11-21

### Added
- Initial release of MCP Clarify (formerly MCP HITL Server)
- `ask_clarification` tool for structured user input elicitation
- Support for multiple choice questions with enum schemas
- Cross-client compatibility (Cursor, Claude Desktop, VS Code)
- Pydantic v1 and v2 compatibility shims
- Adaptive elicitation strategies for different FastMCP versions
- Comprehensive documentation and examples
- GitHub Actions CI/CD workflows
- Community files (CODE_OF_CONDUCT, CONTRIBUTING, SECURITY)
